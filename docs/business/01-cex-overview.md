# CEX 概述

## 基本概念

中心化交易所（CEX）通过链上资产托管模式完成交易与撮合。

**特点：** 交易性能更优；具备监管、AML 等合规能力。

```
法币世界  ←—— CEX 桥梁 ——→  DeFi 世界
```

CEX 提供：法币入金、流动性提供、价格发现、合规服务。

## CEX 与 DEX 对比

| 维度 | 中心化交易所（CEX） | 去中心化交易所（DEX） |
| :--- | :--- | :--- |
| **订单种类** | 订单种类较多，有市价订单和限价订单，及更复杂的订单类型。 | 订单种类较少，目前只支持即时交换。 |
| **交易费用** | 费用较高，依照平台交易手续费和交易的币种。 | 费用较低，费用取决于所在的区块链。 |
| **交易速度** | 速度较快，基于中央基础设施可达成快速交易。 | 速度较慢，每一笔交易记录在区块链网络中，往往导致交易速度慢。 |
| **安全性** | 安全性较低，用户资产由中心化交易所管理，容易成为黑客攻击目标。 | 安全性较高，用户资产存放在个人钱包，钱包私钥由用户个人管理。 |
